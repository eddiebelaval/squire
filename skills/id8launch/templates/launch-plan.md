# Launch Plan Template

Use this template to create a comprehensive launch checklist for your product.

---

## Template

```markdown
# Launch Plan: {Product Name}

**Launch Date:** {YYYY-MM-DD}
**Primary Channel:** {Channel}
**Launch Type:** {Soft / Hard / Continuous}
**Status:** {Planning / Ready / Launched}

---

## Pre-Launch Summary

### Positioning
**For** {target audience}
**who** {have this problem}
**{Product}** is a {category}
**that** {provides this benefit}
**Unlike** {competitor}
**we** {key differentiator}

### Launch Goals
| Goal | Target | Success Metric |
|------|--------|----------------|
| Sign-ups | {N} | Registration count |
| Activation | {N}% | % who complete setup |
| Feedback | {N} | Conversations/responses |

### Budget
| Item | Cost |
|------|------|
| {item} | ${X} |
| **Total** | **${X}** |

---

## T-4 Weeks: Foundation

### Positioning & Messaging
- [ ] Finalize positioning statement
- [ ] Write tagline (2-5 words)
- [ ] Write one-liner (10-15 words)
- [ ] Write value proposition (2-3 sentences)
- [ ] Define 3 key benefits

### Pricing
- [ ] Finalize pricing model
- [ ] Set price points
- [ ] Create pricing page copy
- [ ] Set up payment processing

---

## T-2 Weeks: Assets

### Landing Page
- [ ] Hero section with clear CTA
- [ ] Feature highlights
- [ ] Social proof (if available)
- [ ] Pricing section
- [ ] FAQ section
- [ ] Mobile responsive check
- [ ] Performance optimization

### Visual Assets
- [ ] Product screenshots (5-6)
- [ ] Demo GIF or video
- [ ] Social sharing images
- [ ] Product Hunt assets (if applicable)
  - [ ] Thumbnail (240x240)
  - [ ] Gallery images
  - [ ] Maker comment draft

### Copy Assets
- [ ] Landing page copy
- [ ] Email announcement draft
- [ ] Social media posts (3-5 variants)
- [ ] First comment/story for launches
- [ ] Press kit (if doing outreach)

---

## T-1 Week: Preparation

### Technical
- [ ] Final QA of sign-up flow
- [ ] Load testing (if expecting volume)
- [ ] Error monitoring set up
- [ ] Analytics configured
- [ ] Backup plan for issues

### Community
- [ ] Notify close supporters
- [ ] Schedule social posts
- [ ] Prepare personal outreach messages
- [ ] Join relevant communities (lurk first)

### Content
- [ ] Queue blog post (if applicable)
- [ ] Prepare newsletter draft
- [ ] Schedule tweets/posts

---

## T-1 Day: Final Prep

### Checklist
- [ ] Landing page final review
- [ ] Sign-up flow working perfectly
- [ ] Payment processing tested
- [ ] All links working
- [ ] Social posts scheduled
- [ ] Email ready to send
- [ ] Coffee ready for tomorrow

### Personal Prep
- [ ] Clear calendar for launch day
- [ ] Notifications set up for mentions
- [ ] Response templates ready
- [ ] Rest up (seriously)

---

## Launch Day: Execute

### Early Morning (if PH launch: 12:01am PT)
- [ ] Submit to primary platform
- [ ] Post first comment/story
- [ ] Initial social announcement
- [ ] Send to close supporters

### Morning
- [ ] Monitor for issues
- [ ] Respond to early comments
- [ ] Send launch emails
- [ ] Additional social posts

### Throughout Day
- [ ] Respond to ALL comments within 30 min
- [ ] Track key metrics hourly
- [ ] Share momentum updates
- [ ] Fix any bugs immediately
- [ ] Screenshot milestones

### Evening
- [ ] Thank supporters publicly
- [ ] Compile day's metrics
- [ ] Note learnings and feedback
- [ ] Plan Day 2 activities

---

## T+1 Day: Follow-up

### Immediate
- [ ] Respond to overnight comments
- [ ] Share results/thank you post
- [ ] Send thank you to key supporters
- [ ] Address any issues from Day 1

### Analysis
- [ ] Compile launch metrics
- [ ] Review feedback themes
- [ ] Identify quick wins
- [ ] Document learnings

---

## T+1 Week: Sustain

### Momentum
- [ ] Publish "behind the scenes" content
- [ ] Share user testimonials
- [ ] Continue community engagement
- [ ] Plan next content piece

### Iteration
- [ ] Triage feedback
- [ ] Ship quick fixes
- [ ] Plan feature additions
- [ ] Update roadmap

---

## Channel-Specific Checklists

### If Product Hunt
- [ ] Maker profile complete
- [ ] Hunter lined up (optional)
- [ ] Thumbnail (240x240)
- [ ] Gallery images (5-6)
- [ ] Tagline (60 chars)
- [ ] Description (260 chars)
- [ ] First comment (your story)
- [ ] Topics selected
- [ ] Launch at 12:01am PT
- [ ] Support team notified (don't ask for votes)

### If Hacker News
- [ ] Show HN post drafted
- [ ] Technical angle clear
- [ ] Ready to handle criticism
- [ ] Timing: weekday morning EST

### If Reddit
- [ ] Relevant subreddits identified
- [ ] Rules read for each
- [ ] Non-promotional post drafted
- [ ] Ready to engage authentically

### If Twitter/X
- [ ] Thread drafted
- [ ] Images/GIFs ready
- [ ] Scheduling set up
- [ ] Reply notifications on

- [ ] Paid promotion backup

- [ ] Response templates
- [ ] Rapid iteration plan
- [ ] Community management
```

## Usage Notes

1. Start filling this out 4+ weeks before launch
2. Customize checklists based on your channels
3. Delete sections that don't apply
4. Add items as you discover them
5. Use this as living document through launch
