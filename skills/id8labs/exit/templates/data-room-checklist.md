# Data Room Checklist Template

Use this checklist to prepare your due diligence data room.

---

## Template

```markdown
# Data Room: {Company Name}

**Prepared for:** {Buyer Name / General DD}
**Date Prepared:** {YYYY-MM-DD}
**Last Updated:** {YYYY-MM-DD}
**Access:** {who has access}

---
```

## Usage Notes

1. **Start early** - Building a data room takes weeks, not days
2. **Organize clearly** - Use consistent naming and folder structure
3. **Use a platform** - Google Drive, DocSend, or dedicated VDR
4. **Track access** - Know who's viewing what
5. **Update regularly** - Keep documents current during DD
6. **Watermark sensitive docs** - For confidentiality
7. **Index everything** - Make documents easy to find
