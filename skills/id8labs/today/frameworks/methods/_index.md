# Methods Library Index

This library contains 14 productivity methodologies. Each can be applied to organize your day. Use `/today apply [method]` to activate one.

## Method Categories

### Prioritization Methods
Decide what's important and in what order.
- **Eisenhower Matrix** - Urgent × Important quadrants
- **Must-Should-Could** - Three-tier prioritization
- **1-3-5 Rule** - Size-based prioritization

### Focus Methods
Protect attention and maintain concentration.
- **Pomodoro Technique** - Timed work sprints
- **Eat the Frog** - Hardest task first
- **Time Blocking** - Calendar-based focus

### Capture Methods
Get everything out of your head.
- **GTD (Getting Things Done)** - Full capture system
- **Personal Kanban** - Visual task flow

### Execution Methods
Actually getting things done.
- **Ivy Lee Method** - Six tasks, ranked
- **Two-Minute Rule** - Quick task dispatch
- **Batching** - Group similar tasks

### Energy Methods
Match work to energy levels.
- **Energy Mapping** - Task-energy alignment
- **Weekly Themes** - Day-based focus areas

### Combined Methods
Method combinations for specific contexts.
- **Hybrid Recipes** - Proven method combinations

## Applying Methods

### Single Method
```
/today apply eisenhower
```
Applies Eisenhower Matrix to today's tasks.

### Method Suggestion
```
/today suggest
```
Gets context-aware recommendation.

### Hybrid Recipe
```
/today apply morning-frog-afternoon-batch
```
Uses predefined combination.

## Method Philosophy

**No method is always right.**

The best method depends on:
- Today's task volume
- Your current energy
- Your resistance level
- Time available
- Task type mix

**The goal is doing, not organizing.**

If you spend more time managing tasks than doing them, simplify. Pick a method, use it, adjust tomorrow.

**Any system beats no system.**

Even a simple list beats keeping tasks in your head. Start simple, add complexity only when needed.
