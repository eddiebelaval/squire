## Platform Mastery

### X (Twitter) - 2025 Algorithm Knowledge

**Ranking Weights:**
| Signal | Weight | Strategy |
|--------|--------|----------|
| Quotes | 5x | Create quotable content |
| Bookmarks | 4x | Provide actionable value |
| Replies | 3x | Ask genuine questions |
| Retweets | 2x | Make sharing easy |
| Likes | 1x | Emotional resonance |

**Penalty System:**
- External links in main tweet: -100 (move to reply)
- Excessive hashtags: -30 each beyond 2
- Duplicate/spam content: -50
- Filtered words (free, giveaway, scam): shadowban risk

**Optimal Posting:**
- Character sweet spot: 71-100 characters for max engagement
- Rich media boost: +2x for original image/video
- Conversation starters outperform announcements
- Questions in opening hook drive replies
- Thread format for complex topics (3-7 tweets optimal)

### LinkedIn - Professional Engagement

**Algorithm Priorities:**
- Dwell time (longer reads rank higher)
- Comments over reactions
- Early engagement (first 60 minutes critical)
- Native content over external links

**Content Patterns That Work:**
```
Hook Line (problem statement)

[3-5 short paragraphs with line breaks]

[Call to engagement - question or reflection]

---
[Hashtags at bottom, max 3-5]
```

**Engagement Rules:**
- First-person stories outperform advice
- Vulnerable/failure content gets high engagement
- Carousels perform 3x better than text posts
- Polls drive algorithm visibility
- Comment within first hour on your own posts

### Instagram - Visual-First Strategy

**Feed Post Optimization:**
- Carousel posts: 3x engagement vs. single image
- First slide = scroll-stopper hook
- Save-worthy content (tips, guides) performs best
- Caption length: 150-300 words optimal
- CTA in caption + first comment

**Stories Strategy:**
- Post 3-7 stories per day
- Use polls/questions for engagement
- Behind-the-scenes content builds connection
- Link stickers for driving traffic

**Reels Algorithm:**
- First 3 seconds = hook or scroll
- 15-30 second sweet spot
- Trending audio = discovery boost
- Text overlays for silent viewing
- Post between 7-9 PM local time

### TikTok - Discovery Engine

**Algorithm Factors:**
- Completion rate is king
- Rewatch signals boost

**Content Framework:**
- Hook in first 1 second
- Pattern interrupts maintain attention
- Call to action at end
- Duet/stitch for commentary content

## Content Creation Framework

### The SPEAR Method for Posts

**S - Stop the Scroll**
- Pattern interrupt in first line
- Controversy, curiosity, or counterintuitive hook
- Numbers and specifics outperform vague claims

**P - Promise Value**
- What will the reader gain?
- Be specific about the benefit
- Match promise to platform expectations

**E - Elaborate Concisely**
- Deliver on the promise
- Use white space and short sentences
- Make it scannable

**A - Amplify with Media**
- Add visual support
- Screenshots, charts, or original images
- Platform-appropriate format

**R - Request Action**
- Clear but non-pushy CTA
- Questions drive more engagement than commands
- Make the action easy

**Story/Experience:**

## Reply & Engagement Strategy

### Reply Tone Calibration

**Positive Comments:**
- Acknowledge specifically what they said
- Add value with a follow-up insight
- Ask a question to continue conversation
- Use their name if appropriate

**Neutral Comments:**
- Engage genuinely with their point
- Provide additional context
- Bridge to conversation continuation

**Critical Comments:**
- Never be defensive
- Acknowledge valid points
- Clarify misunderstandings calmly
- Offer to continue in DMs if heated
- Kill with professionalism, not snark

**Troll Comments:**
- Assess if genuine criticism or trolling
- If trolling: brief acknowledgment or ignore
- Never feed with emotional response
- Block/mute persistent harassers

### Reply Templates

**Thanking a Supporter:**
```
Appreciate you [name]! [Specific acknowledgment of their point].

[Add value or insight]

[Optional: question to continue conversation]
```

**Addressing Criticism:**
```
Thanks for the feedback, [name].

[Acknowledge their point].

Here's some context: [explanation without excuse]

Happy to discuss further - feel free to DM me.
```

**Building on Discussion:**
```
Great point about [their topic].

I'd add [your insight] because [reasoning].

Have you tried [relevant suggestion]?
```

## Brand Voice Framework

### Voice Calibration Questions

Before creating content, establish:

1. **Tone:** Professional, casual, playful, authoritative?
2. **Vocabulary:** Technical jargon or plain language?
3. **Emoji usage:** Heavy, light, or none?
4. **Humor level:** Serious, witty, or meme-heavy?
5. **Engagement style:** Formal responses or conversational?

### Voice Consistency Checklist

- [ ] Would this sound like the same person as the last post?
- [ ] Does the language match audience expectations?
- [ ] Is the tone appropriate for the topic?
- [ ] Are branded terms used correctly?
- [ ] Does it feel authentic, not performative?

## Crisis Communication Protocol

When facing social media crisis:

1. **Pause scheduled content** - Don't appear tone-deaf
2. **Assess the situation** - Understand scope and sentiment
3. **Respond promptly** - Silence looks like guilt
4. **Acknowledge** - Don't deny or deflect
5. **Take ownership** - If at fault, own it
6. **Outline action** - What you're doing about it
7. **Follow up** - Update when resolved

### Crisis Response Template

```
We hear you. [Acknowledge the issue directly]

[What happened/what you know]

[What you're doing about it]

We'll share updates as we have them.

[Optional: where to reach you directly]
```
